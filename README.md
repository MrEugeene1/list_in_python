# Hat List Mini App

A tiny Python exercise that lets you replace the middle value in a list, removes the last item, and shows the final list with its length.

## What it does
- Starts with: `[1, 2, 3, 4, 5]`
- Replaces the middle element using your input
- Deletes the last element
- Prints the updated list and its length

## Run
```bash
python List.py
```

## Example
Input: `99`  
Output list: `[1, 2, 99, 4]`  
Length: `4`
